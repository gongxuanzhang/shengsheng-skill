# CLAUDE.md 新增章节模板

当需要在 CLAUDE.md 中新增内容时，使用以下对应模板。

---

## 设计说明条目模板

```markdown
### {决策名称}

**问题**：{一句话描述要解决的问题}

**方案**：{选择的方案及关键实现}

{代码示例（如有）}

**覆盖场景**：
- {场景 1}
- {场景 2}
```

---

## API 端点模板

```markdown
#### `{METHOD} {path}` — {一句话说明}

{详细说明}

**请求体：**
\`\`\`json
{示例 JSON}
\`\`\`

**响应体：**
\`\`\`json
{示例 JSON}
\`\`\`
```

---

## 代码规范条目模板

```markdown
- **{规范名称}统一使用 `{类/方法名}`**，不要在各处重复实现 {什么逻辑}
```

---

## 故障排查条目模板

```markdown
| {现象} | {排查方向} |
```

---

## 项目结构文件条目模板

```markdown
│   ├── {FileName.java}               # {一句话说明文件职责}
```

---

## 配置项模板

```markdown
**配置项（application.yml → `app.{section}`）**：

\`\`\`yaml
app:
  {section}:
    {key}: {default-value}    # {说明}
\`\`\`
```
